1. Set up the question paper & answer key

Add each question with its correct answer (MCQ/numeric) or marking guide (short/essay). You can add as many as you like, then save the paper.

2. Upload the question paper (image)

One image with several numbered questions. The system reads them all. Set a default type & marks (you can refine later).

3. Upload the answer key (image)

One image with the correct answer for each numbered question. Matched to the questions above by number.

3b. Review & edit questions and marks

Marks read from the paper are shown; missing ones use the default. Edit any marks here, then Save marks.

4. Upload a student's answer script (image)

One image with all the student's numbered answers (and their name/ID on the page). Each answer is matched to its question by number and graded.
